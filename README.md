# Pydle
Pydle is based on the game Wordle by The New York Times and written purely in python. <br>
This project is only made for fun :)

## How it works
Same as Wordle, guess the word in five attempts. There are indicators if the word you guessed is incorrect or not.<br>
### Example:
```
# # E* # L
```
\# = Incorrect<br>
E* = Correct, but in wrong the order<br>
L = Correct<br>

And that's it! I will be planning to add more features if I'm not too lazy .-.<br>
